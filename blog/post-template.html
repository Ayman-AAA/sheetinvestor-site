How to Track Dividend Stocks in Google Sheets

This guide shows step-by-step how to use Dividend Toolkit functions to track dividend income, yields, and project future payments.

Install

Install the add-on from the Google Workspace Marketplace.

Basic example

=latestDividend("AAPL")

Use the function in any cell to fetch the most recent dividend amount.
